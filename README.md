# llm-context-packer: The Micro-CLI for Prompt Engineering
> Instantly flatten your local codebase into a token-optimized Markdown string for ChatGPT/Claude.

## ⚡ Quick Start
`python llm_packer.py /path/to/project`

## 🧠 Why?
Built to solve the "context stuffing" friction. It converts complex directory structures into a single, structured prompt that LLMs can actually reason about.
